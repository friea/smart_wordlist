## Smart Wordlist
Anahtar kelimeler girildiğinde onlarla bağlantılı sözcük ve sayıları bulan ve bir wordlist oluşturan masaüstü uygulaması.
Zero Shot Learning tabanlı GPT-2 dil modeli ile oluşturulmuştur.
